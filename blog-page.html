2. How will you select HTML elements using DOM? Name the DOM Methods.

Solution: HTML can be selected many ways but mainly 5 ways are used to get the value from the elements and it used the ID to get the data from html elements with helps of javascript. Mainly DOM helps to connected the html section using id then we get data from it.

1. getElementsByTagName()

2. getElementsByClassName()

3. getElementById()

4. querySelector()

5. querySelectorAll()

3. What is Event Bubble?

Solution: when any event enter another event and the first one was target for event then the first inner part will be target then the outer side or the parents tag also be targeted. That is called Event Bubble.

4. What is a callback function? Why will you use it?

Solution: when any type of value will called any function then the parameter will reused again inside of this function which can also called another function for reuse, that is called callback function. reduce the coding time and maintain the usability of good product with good output.
